sublime 安装MarkDown插件

安装package control

  • Ctrl+Shift+p
  • 输入install Package Control
  • Ctrl+Shif+p
  • 输入pcip(选择 Package Control:Install Package)
  • 输入mmp(选择 Markon real-time markdown preview)

安装markdown preview

  • Ctrl+Shift+p
  • 输入pcip(选择 Package Control:Install Package)
  • 输入Markdown Preview

使用markdown插件

  • ctrl+Shift+p
  • 输入mpp(选择 Markdown Preview:preview in browser)
  • 选择git风格或者markdown风格

保存markdown为HTML

  • Ctrl+Shift+p
  • 输入mpsth(选择 Markdown Preview:save to HTML)
  • 选择git风格或markdown风格
